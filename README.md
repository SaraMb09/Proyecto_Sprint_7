## Análisis Exploratorio de Vehículos Usados

# Descripción del proyecto
Este proyecto se basa en el desarrollo de una aplicación web interactiva utilizando Streamlit, orientada a llevar a cabo un análisis exploratorio de datos (EDA) a apartir de un conjunto de anuncios de vehículos usados en Estados Unidos.
La aplicación brinda a los usuarios la posibilidad de examinar visualmente el comportamiento del mercado automotriz, detectar tendencias en los precios y estudiar la relación entre relevantes como el precio, el año de fabricación del vehículo y el kilometraje recorrido.

## Funciones principales
La aplicación web proporciona las siguientes funciones:
Histograma interactivo de precios: esto permite visualizar la distribución de precios de los vehículos y detectar valores atípicos.
Gráfico de disperción (precio vs kilometraje): Muestra la relación entre el precio del vehículo y su kilometraje, ayudando a identificar tendencias y posibles anomalías.
Interactividad mediante casillas de verificación: El usuario puede seleccionar qué gráficos desea visualizar usando checkbox, lo cual facilita la exploración dinámica de los datos.

## Tecnologías usadas
Python
Pandas: manipulación y análisis de datos.
Streamlit: desarrollo de la aplicación web.
Plotly Express: visualización de datos.

## Objetivo del proyecto
Demostrar las habilidades en análisis exploratorio de datos y visualización interactiva, así como la capacidad de crear una aplicación web sencilla para presentar resultados de manera clara e intuitiva.
